# PDF 도구

PDF 파일을 병합하고 분리할 수 있는 간단한 도구입니다.

## 다운로드 및 설치

1. `dist` 폴더에서 `PDF 도구.zip` 파일을 다운로드합니다.
2. 압축을 해제합니다.
3. `PDF 도구.exe` 파일을 실행합니다.

## Windows에서 실행 시 주의사항

### 1. 파일 차단 해제 방법

1. `PDF 도구.exe` 파일을 마우스 오른쪽 버튼으로 클릭합니다.
2. '속성'을 선택합니다.
3. '일반' 탭에서 '차단 해제' 체크박스를 선택합니다.
4. '적용'을 클릭한 후 '확인'을 클릭합니다.

### 2. Windows Defender 예외 처리 방법

1. Windows 보안을 엽니다.
2. '바이러스 및 위협 방지'를 선택합니다.
3. '바이러스 및 위협 방지 설정'에서 '설정 관리'를 클릭합니다.
4. '제외 추가 또는 제거'를 선택합니다.
5. '제외 추가'를 클릭하고 `PDF 도구.exe` 파일을 선택합니다.

### 3. 파일 무결성 확인

실행 파일의 SHA-256 해시 값:

```
[실행 파일의 SHA-256 해시 값이 여기에 표시됩니다]
```

## 사용 방법

### PDF 병합하기

1. 'PDF 파일 선택' 버튼을 클릭하거나 파일을 드래그앤드롭하여 병합할 PDF 파일들을 선택합니다.
2. '위로'와 '아래로' 버튼으로 파일 순서를 조정할 수 있습니다.
3. 'PDF 병합하기' 버튼을 클릭하고 저장할 위치를 선택합니다.

### PDF 분리하기

1. 'PDF 파일 선택' 버튼을 클릭하여 분리할 PDF 파일을 선택합니다.
2. 'PDF 분리하기' 버튼을 클릭합니다.
3. 분리할 페이지 범위를 입력합니다 (예: 1-3,5,7-9).
4. 분리된 파일들이 저장될 폴더를 선택합니다.
